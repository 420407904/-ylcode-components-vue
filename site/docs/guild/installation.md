## 使用包管理器

建议您使用包管理器 (NPM, Yarn, pnpm) 安装 kitty-plus, 然后您就可以使用打包工具，例如 Vite 和 webpack

```
# 选择一个你喜欢的包管理器

# NPM
$ npm install kitty-plus --save

# Yarn
$ yarn add kitty-plus

# pnpm
$ pnpm install kitty-plus
```
